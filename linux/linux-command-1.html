查看已被删除但未释放的文件

lsof (list opened files)

注意
如果使用 ( lsof -n |grep deleted ) 命令查看时，可能会出现以下错误
lsof: no pwd entry for UID 5701

错误原因
1.进程在用户被删除之前启动，并且从那以后一直在运行
2.进程在容器内被启动

解决方法
-w 是指忽略这些错误
